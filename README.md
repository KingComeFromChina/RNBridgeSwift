# RNBridgeSwift
RN环境搭建和原生交互
#[简介](http://www.jianshu.com/p/1537bb431d12)
`RN`全称`React Native`，是`Facebook`公司推出的一款能够跨平台开发的框架，其能够使用`js`来达到`安卓端`和`iOS端`共用一套代码开发的目的。` React Native `的核心设计理念是：即拥有`React`的开发效率，又能拥有`Native`的体验。
